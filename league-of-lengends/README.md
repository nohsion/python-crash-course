# LOL 디스코드 봇

## 소개
이 봇은 디스코드에서 사용할 수 있는 봇입니다. 이 봇은 리그 오브 레전드와 관련된 정보를 제공합니다.

## 명령어
- `!롤 <소환사명#태그>`: 소환사의 전적을 확인합니다.
- `!롤챔 <챔피언명>`: 챔피언의 정보를 확인합니다.
